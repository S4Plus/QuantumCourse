# 关于使用
导入simplify_pack.py库之后，调用simplify_with_rule(list,list_rule)函数即可实现子图查找以及替换
可以参考示例example.py

# 关于rule文件
目前已提供三个rule文件，可以参考其格式
以CNOT门为例,语句（用*代替空格）:
cx*2*12*13*
其中cx表示这是一个CNOT门，2表示这是一个两比特门，12，13为两比特门的位置，其中第一个为控制位，第二个为被控制位
如果需要自定义门（比如说cxxx），使用类似语法即可：
cxxx*4*Q1*Q2*Q3*Q4*
这可以满足块替换的需求(把特定的量子门组合成一个物理上方便实现的自定义组合门）
自定义门的一个例子可以参见rule4.txt(Toffoli门)

#目前支持读入的门操作（但简化结果支持自定义门，只要写好规则即可，见上一段）
